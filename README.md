## FinSec Lab

Security research focused on Android — reverse engineering, mobile RASP, device-identity, and anti-fingerprinting.

- Mobile app security, instrumentation, and integrity-check analysis
- Builds:
  - [**DeviceAnon**](https://github.com/Finsec-lab/DeviceAnon) — device-identity spoofer (Magisk module, 38 real-device profiles, Android 5–16)
  - [**Revofix**](https://github.com/Finsec-lab/Revofix) — one-tap fix to run Revolut on rooted Android (Magisk module, per-process scope)
- Interests: Frida, Magisk/Zygisk, KernelSU, LSPosed, Play Integrity, request signing, native anti-tamper

Working in the open. Issues, repros and counter-examples welcome.
